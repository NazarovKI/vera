# KPI Маркетолога
Задача: обеспечить необходимое количество целевых лидов
- соотношение новых / цевевых
- именно целевые лиды нужно отслеживать при формировании отчёта - план/факт по лидам.
Задача: обеспечить необходимое количество заявок, которое должен получать отдел продаж ежедневно. Распределять трафик равномерно в течении месяца, чтобы  небыло перегрузки на менеджеров отдела продаж.

Подссчитывать:
- кол-во новых
- кол-во по конкретному каналу
- кол-во целевых
- конверсия из новых в целевые формула
- выполнение плана по целевым заявкам план/факт.

KPI Менеджера
- "дожимать" клиентов после выставления счёта на оплату
- контролировать оплату от клиента
- следить за конверсией в продажу по каждому менеджеру.
