# Настроить автоматическую обработку лидов
- [ ] Связать с [](описание%20процессов.md)
- [ ] Отредактировать в соответствии с текущим состоянием
- [ ] Связать с [opisanie_processov_kompanii](opisanie_processov_kompanii.md)

Сейчас существуют [](проблемы.md)
Отлично! Мы можем настроить автоматический сценарий обработки лидов в Bitrix24, чтобы собирать информацию по этим вопросам и квалифицировать лиды на автоматизированном этапе. Вот как это можно реализовать:  
  
**🛠 Настройка автоматического сценария обработки лидов в Bitrix24**  поможет решить некоторые проблемы
  
**1. **Создайте необходимые поля в карточке лида****  
Перед началом работы добавьте в карточку лида следующие пользовательские поля (в разделе **CRM → Настройки → Поля лида**):  
  
- **Тип объекта** (список): балкон, лоджия, оба  
- **Размеры проема** (строка): длина x ширина x высота  
- **Этаж** (число)  
- **Тип дома** (список): панельный, кирпичный, новостройка  
- **Планируемая дата остекления** (дата)  
- **Бюджет** (число)  
- **Собственник?** (да/нет)  
- **Обращались к каким компаниям?** (строка)  
  
  
**2. **Настройте автоматический сценарий через Bitrix24 (или с помощью интеграции с CoPilot)****  
В Bitrix24 вы можете использовать **автоматизацию** или **CoPilot** для запуска сценариев. Вот пример сценария:  
  
**🔄 Сценарий: "Квалификация лида по остеклению балконов"**  
  
1. **Событие запуска:**  
- Новый лид  
- Или: лид перешел в статус "Новый"  
  
2. **Действие 1: Отправить автоматическое сообщение**  
- Сообщение:  
    

Здравствуйте, меня зовут [Ваше имя], мы с вами связались по поводу остекления балконов. Вы всё ещё рассматриваете этот вопрос?  
  
     Чтобы мы могли максимально точно рассчитать стоимость и сроки, прошу уточнить:  
  
     1. Что именно вы хотите остеклить — балкон, лоджию или оба?  
     2. Каковы примерные размеры? (длина, ширина, высота)  
     3. На каком этаже находится квартира? Какой тип дома? (панельный, кирпичный, новостройка)  
     4. Когда вы планируете провести остекление? (сейчас, через месяц, через полгода?)  
     5. Есть ли у вас ориентировочный бюджет? (например, до 50 тыс. руб.)  
     6. Вы являетесь собственником квартиры?  
     7. Были ли уже согласования с управляющей компанией по поводу остекления?  
     8. Уже обращались к другим компаниям?  
     9. Что именно вас интересует в остеклении? (цена, материалы, сроки, гарантия)  
  
     Мы ответим на все ваши вопросы и поможем с выбором. Спасибо!

  
10. **Действие 2: Установить статус лида**  
- Установить статус: "В ожидании ответа"  
  
4. **Действие 3: Настроить напоминания**  
- Если клиент не ответил в течение 1 дня → отправить напоминание  
- Если клиент не ответил в течение 3 дней → установить статус "Неактивный"  
  
  
**3. **Настройка статусов лида****  
Создайте следующие статусы в разделе **CRM → Настройки → Статусы лида**:  
  
- Новый  
- В ожидании ответа  
- Квалифицирован  
- Неактивный  
- Закрыт  
  
  
**4. **Интеграция с CoPilot (если доступна)****  
Если вы используете Bitrix24 CoPilot, вы можете создать **автоматический сценарий общения**:  
  
- Настройте **шаблон сообщения** с вопросами (как выше)  
- Укажите, что если клиент отвечает на вопросы, сценарий продолжает работу  
- Если клиент не отвечает — сценарий переходит в статус "Неактивный"  
    
**✅ **Как настроить поля в CRM (карточке лида) в Bitrix24****  
  
****1. Перейдите в настройки полей лида****  
- Откройте Bitrix24 → **CRM** → **Настройки** → **Поля лида**  
i[/i]  
  
****2. Добавьте следующие пользовательские поля****  
  
| Название поля                      | Тип поля            | Дополнительные настройки                                                   |     |
| ---------------------------------- | ------------------- | -------------------------------------------------------------------------- | --- |
| **Тип объекта**                    | Список (один выбор) | Варианты: _балкон, лоджия, оба_                                            |     |
| **Размеры объекта**                | Текстовое поле      | Подсказка: _например: 3,5х1,8х2,5 м_                                       |     |
| **Этаж**                           | Число               | Минимум: 1, Максимум: 20                                                   |     |
| **Тип дома**                       | Список (один выбор) | Варианты: _панельный, кирпичный, новостройка, монолитный_                  |     |
| **Планируемая дата остекления**    | Дата                | Можно оставить пустым — для приоритизации                                  |     |
| **Бюджет (руб.)**                  | Число               | Минимум: 0, Максимум: 500000                                               |     |
| **Собственник?**                   | Да/Нет              | Переключатель                                                              |     |
| **Согласование с УК?**             | Да/Нет              | Переключатель                                                              |     |
| **Обращались к другим компаниям?** | Да/Нет              | Переключатель                                                              |     |
| **Что вас интересует?**            | Множественный выбор | Варианты: _цена, материалы, сроки, гарантия, дизайн, установка, утепление_ |     |
  💡 **Совет:** Все поля сделайте **обязательными для заполнения** при создании лида — это повысит качество данных.

****3. Расположите поля удобно****  
- Перетащите поля в карточке лида в логичном порядке:  
**Сначала — объект, размеры, этаж, тип дома** → потом **бюджет, дата, собственник** → в конце — **вопросы про конкурентов и интересы**.  
  
****4. Добавьте подсказки (опционально, но полезно)****  
- Для каждого поля нажмите «Редактировать» → в поле **«Подсказка»** введите пример:  
_Пример: «3,5х1,8 м»_  
_Пример: «до 60 000 руб.»_  
Это поможет менеджерам и клиентам заполнять поля правильно.  
  
****5. Проверьте результат****  
- Создайте тестовый лид → убедитесь, что все поля отображаются и сохраняются.  
- Убедитесь, что поля видны в списке лидов (в таблице) — для быстрой фильтрации.  
  
**🚀 Бонус: Как использовать эти поля?**  
- **Фильтрация:** Ищите лиды по бюджету, типу дома, собственнику — быстро находите «горячие» заявки.  
- **Отчёты:** Создайте отчёт «Лиды с бюджетом > 50 тыс. и собственником» — это ваша золотая аудитория.  
- **Автоматизация:** Эти поля будут использоваться в сценарии, который мы настраивали ранее — для автоматического определения квалификации.
